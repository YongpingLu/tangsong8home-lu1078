# 唐宋八大家

一款以「唐宋八大家」为主题的微信小程序：按出生年代排成时间线，品读韩愈、柳宗元、欧阳修、苏洵、曾巩、王安石、苏轼、苏辙八位文学宗师的 **188 首**代表诗词文，并集成央视《宗师列传·唐宋八大家》纪录片观看入口与「跟读 / 回放自评」练习功能。

## 功能特性

- **时间线首页**：八位作者按生卒年排列，形成时间轴视觉；点击进入作者介绍页。
- **作者介绍**：生平简介、字号、朝代等信息。
- **诗词品读**：共 188 首真实代表作，含诗、词、文，附注释与译文；支持展开/收起译文、上一篇/下一篇翻阅。
- **央视视频入口**：每位作者关联央视《宗师列传·唐宋八大家》正片；韩愈、欧阳修分上/下集，提供分集复制观看链接。
- **跟读练习**：原生录音（`RecorderManager`）录制朗读，本机保存不上传。
- **回放自评**：录音回放支持**播放 / 暂停 / 从暂停处续播**，单实例播放杜绝声音重叠。
- **声纹授权协议**：录音功能前置隐私协议页，符合合规要求。

## 快速开始

1. 安装并打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)。
2. 「导入项目」→ 选择本仓库根目录（**直接包含 `app.json` 的目录**）。
3. AppID 可选择「测试号」或填入你自己的小程序 AppID。
4. 编译预览即可运行。

## 技术说明

- 原生微信小程序框架（WXML / WXSS / JS），无第三方构建依赖。
- 录音使用 `wx.getRecorderManager()`；回放使用 `wx.createInnerAudioContext()`（单实例，`play/pause/resume/stop` 精确控制进度）。
- 视频不在小程序内直接播放，改为提供央视官方观看链接（个人主体不支持 web-view，兼顾版权与审核合规）。
- 主题配色：宣纸底 `#f3ece0` + 深红 `#7a1f1f`，古典雅致。
